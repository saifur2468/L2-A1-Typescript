# TypeScript Assignment

## 📌 Project Overview

এই প্রজেক্টটি TypeScript-এর বিভিন্ন গুরুত্বপূর্ণ কনসেপ্ট যেমন function, interface, type alias, generics, OOP, inheritance, polymorphism এবং utility type ব্যবহার করে তৈরি করা হয়েছে।

এই অ্যাসাইনমেন্টের মূল উদ্দেশ্য হলো TypeScript-এর type safety এবং modern JavaScript feature গুলো প্র্যাকটিস করা।

---

# 🚀 Features

* Type-safe Function ব্যবহার
* Array Method Practice
* Interface এবং Type Alias
* Union & Intersection Type
* Generic Function
* OOP Concepts
* Inheritance & Polymorphism
* Utility Types
* Clean & Reusable Code
